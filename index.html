C'est quoi ?

  • Transitions animées en CSS
  • Sans JavaScript !
  • Spécification en cours de développement (Working Draft)
  • Ne pas confondre avec les Animations CSS !

À quoi ça sert ?

  • Fonctionnalité non critique
  • Ésthétique, Design, "Visual rewards"
  • Meilleur feedback visuel, meilleure expérience usager
  • Remplace JavaScript (jQuery.animate)
    • Code plus propre, maintenance aisée
    • Plus léger, intéressant pour un site mobile
    • Peut être généré par un outil

Ma première transition CSS

.logo {
  width: 100px;  height: 100px;
  background: url(html5-logo.svg) 50% 50% no-repeat;
  background-size: 80px;
  -webkit-transition-duration: 1s;
     -moz-transition-duration: 1s;
      -ms-transition-duration: 1s;
       -o-transition-duration: 1s;
          transition-duration: 1s;
}
.logo:hover {
  width:700px;
  background-color: silver;
}

Propriétés animables

  • Propriétés numériques
    • margin, padding, width, height...
    • opacity
  • Couleurs
  • CSS3
    • box-shadow, text-shadow
    • Gradients
    • CSS Transforms: translate, rotate, scale, skew

Déclencheurs / triggers

  • Sélecteurs de pseudo-classe
    • :hover
    • :focus
    • :enabled, :disabled
    • :checked
  • Media Queries
  • JavaScript (changement de classe CSS)
